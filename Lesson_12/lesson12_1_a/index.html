Write a function which will:

  1. Accept array of number
  2. Output (via console.log()) array with all numbers increased by 1

Example:

const increasedArray = increaseNumbersByOne([0, 1, 2]);
console.log(increasedArray); // Should output [1, 2, 3]
const increasedArray = increaseNumbersByOne([10, 15, 23]);
console.log(increasedArray); // Should output [11, 16, 24]
